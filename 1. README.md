# Angular 21 Enterprise Architecture Guide

This repository demonstrates how to build a **clean, scalable, and maintainable Angular 21 application** using modern best practices.

## 🧠 What You'll Learn

- Feature-based architecture (vertical slicing)
- Smart vs Presentational components
- Enterprise folder structure
- Shared UI and Core layers
- Signal-based state management
- Angular standalone components & routing

---

## 🏗️ Architecture Principles

This project follows:

- **Feature-first structure**
- **Separation of concerns**
- **Route-based lazy loading**
- **Standalone Angular APIs**
- **Reusable UI components**
